函数

函数定义

C 语言中函数的定义形式一般如下：

returnType functionName (parameterType parameter) {
  functionBody
}
1
2
3

在 C 语言中，函数由函数头和函数体组成。有以下部分：

  • 返回类型：返回值的数据类型
  • 函数名：函数的实际名称，函数名和参数列表一起构成了函数签名
  • 参数：函数的参数。参数分为实参和形参，实参指的是实际传入的变量或值，形参指的是函数所定义的参数
  • 函数体：定义函数所要执行的语句
